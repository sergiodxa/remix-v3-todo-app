# Remix v3 Todo App

This is a simple Remix v3 Todo application built with Bun.

## How to Run

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.3.0. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
